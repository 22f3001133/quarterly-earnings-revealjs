Return on Equity (ROE)

\[ ROE = \frac{\text{Net Income}}{\text{Shareholders' Equity}} \]

Net Interest Margin (NIM)

\[ NIM = \frac{\text{Interest Income} - \text{Interest Expense}}{\text{Average Earning Assets}} \]
Explain that ROE measures profitability relative to equity and NIM reflects the efficiency of interest-earning operations.

import pandas as pd

def compute_qoq_growth(df):
    """
    df columns: ['quarter', 'revenue']
    Returns df with an added 'qoq_growth' percentage column.
    """
    df = df.sort_values('quarter')
    df['qoq_growth'] = df['revenue'].pct_change() * 100
    return df

data = pd.DataFrame({
    "quarter": ["Q1", "Q2", "Q3", "Q4"],
    "revenue": [95.0, 100.0, 110.0, 120.0]
})

result = compute_qoq_growth(data)
print(result)
This Python snippet demonstrates a small analytics step that computes quarter-on-quarter revenue growth using pandas.
